Full Stack Developer

        Full Stack Developer คือ ผู้พัฒนาซอฟต์แวร์ทั้งส่วนที่ติดต่อกับผู้ใช้งาน และส่วนระบบหลังบ้าน หรือก็คือ เป็นทั้ง  Front End Developer  และ  Back End Developer  ในคน ๆ เดียวกันนั่นเอง จริง ๆ แล้วในส่วนของการดูแลหน้าบ้านและหลังบ้านนั้น ก็จะมีตำแหน่ง  Front End Developer  และ  Back End Developer  แยกตัวออกมาอยู่แล้ว แต่ความสำคัญของตำแหน่ง  Full Stack Developer   นั่นก็คือคนกลางที่จะคอยช่วยพัฒนาหรือประสานให้ทั้งหน้าบ้านและหลังบ้านทำงานร่วมกันได้อย่างไม่ติดขัด ซึ่งหากขาด  Full Stack Developer  ไป ก็อาจทำให้โปรเจกต์นั้น ๆ สะดุดหรือพบเจอปัญหาขึ้นได้

Hard skill

1. มีความรู้ด้าน  HTML, CSS และ JavaScript 
2. มีความถนัดและเชี่ยวชาญด้าน Front End Framework เช่น  ReactJS หรือ Angu 
3. มีความเข้าใจใน ภาษาด้าน  Programming เช่น Ruby, PHP หรือ Pyt  
4. มีความเชี่ยวชาญด้าน  Version Control
5. อัดแน่นด้วยประสบการณ์ทางด้านการวางแผน ออกแบบ ติดตั้ง ควบคุม และดูแลรักษาระบบฐานข้อมูล
6. มีความเข้าใจในภาพรวมของโครงสร้างการออกแบบเว็บไซต์
7. มีความรู้สำหรับเรื่องความปลอดภัยบนเว็บไซต์
